로컬에 남기고
브라우저에서 정리하는
개인 가계부 대시보드

Donggri Ledger는 계좌, 현금, 카드 결제 예정, 예산, 투자 현황을 하나의 흐름으로 이어 주는 로컬 실행형 가계부 웹앱입니다. 복잡한 클라우드 가입 없이 바로 띄워서, 내 PC 안에서 재무 기록을 정리하는 데 초점을 맞췄습니다.

월별 현금 흐름과 투자 요약을 한 흐름으로

거래 기록과 카드 예정 결제, 포트폴리오 수익률을 분리하지 않고 같은 대시보드 계열 안에서 이어서 봅니다.

01

데이터는 로컬에, 화면은 브라우저에

FastAPI 서버를 내 PC에서 띄우고 브라우저 UI로 접속하는 구조라서, 사용 흐름은 편하지만 저장은 로컬 중심입니다.

02

카드 결제 예정까지 월 흐름에 반영

카드 자산과 결제일, 출금 계좌를 연결해 실제 현금 흐름과 예정 결제를 함께 점검할 수 있습니다.

03

가계부와 투자 현황을 분리하지 않음

예산, 고정지출, 투자 자산과 수익률을 같은 앱에서 확인해서 월말 정리 동선을 짧게 유지합니다.

Workflow

복잡한 초기 세팅보다, 매달 다시 열어 쓰기 쉬운 흐름

이 앱은 회계 시스템처럼 입력 단계를 늘리기보다, 자산과 결제 수단을 먼저 정리한 뒤 월별 거래를 빠르게 기록하고 요약 카드를 통해 다시 점검하는 흐름에 맞춰져 있습니다.

1. 자산과 카드 연결

은행, 현금, 카드, 투자 계좌를 등록하고 카드 결제 계좌를 지정해 기본 구조를 만듭니다.

2. 월별 거래 입력

수입과 지출, 카드 사용분을 기록하면서 이번 달 현금 흐름과 예정 결제를 함께 누적합니다.

3. 예산과 투자 점검

예산 사용률, 고정지출, 투자 수익률을 나란히 확인해 한 달 마감에 필요한 판단을 빠르게 끝냅니다.

Runtime capture

실행 화면 기준으로 어떤 정보가 보이는지 바로 확인할 수 있습니다

아래 이미지는 실제 로컬 실행 환경에서 캡처한 화면입니다. 첫 화면은 월별 가계부와 카드 결제 예정 요약, 두 번째 화면은 같은 앱 안에서 이어지는 투자 포트폴리오 탭입니다.

STEP 1 가계부 대시보드에서 월 흐름 확인

가용 현금, 카드 결제 예정, 자산별 잔액, 최근 거래가 같은 구간에 배치됩니다.

STEP 2 투자 탭에서 수익률까지 이어 보기

투자 원금, 평가금, 수익률과 종목별 평가 현황을 별도 서비스 없이 같은 앱 안에서 확인합니다.

Search intent

FAQ

자주 묻는 질문

Donggri Ledger는 클라우드 서비스인가요?

아니요. 기본 데이터는 로컬 SQLite에 저장되고, 사용자는 자기 PC에서 FastAPI 서버를 띄운 뒤 브라우저로 접속해 사용합니다.

카드 결제 예정 금액도 따로 관리할 수 있나요?

가능합니다. 카드 자산과 결제일, 출금 계좌를 지정해 월별 카드 결제 예정 금액을 상단 요약 카드에서 확인할 수 있습니다.

투자 자산까지 꼭 같이 넣어야 하나요?

필수는 아니지만, 투자 계좌를 등록해 두면 전체 재무 흐름과 포트폴리오 수익률을 같은 앱 안에서 이어서 볼 수 있습니다.
