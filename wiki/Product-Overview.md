# Product Overview

## What Donggri Ledger Is

Donggri Ledger는 생활비 기록만 하는 단순 가계부가 아니라, 자산 흐름과 카드 결제예정, 예산, 투자 포트폴리오를 함께 보는 개인 재무 관리 앱입니다.

핵심 목표는 아래 세 가지입니다.

- 소비 기록과 실제 자산 흐름을 다르게 다루기
- 카드 결제예정까지 포함한 현실적인 현금 흐름 보여주기
- 투자 자산을 별도 앱이 아니라 전체 자산 흐름 안에서 보기

## Target User

이 제품은 아래 사용자에게 맞춰져 있습니다.

- 카드 사용 비중이 높은 개인 사용자
- 고정지출과 예산을 함께 관리하고 싶은 사용자
- 주식, 코인, ETF, 펀드를 생활비 흐름과 함께 보고 싶은 사용자
- 로컬 파일 기반 앱보다 웹 기반 접근성과 백업 편의성을 선호하는 사용자

## Product Principles

### 1. 입력은 단순하게

거래 입력 과정은 짧고 빠르게 유지합니다.  
사용자가 카드 사용일, 결제일, 실제 출금 로직을 모두 직접 계산하지 않도록 자산 모델에서 처리하는 방향을 유지합니다.

### 2. 화면은 생활 흐름 중심으로

월별 수입/지출 합계만 보여주는 게 아니라, 카드 결제예정, 고정지출, 예산 사용률, 투자 평가금까지 함께 이어지도록 설계합니다.

### 3. 운영은 서비스형으로

집 PC를 켜두는 로컬 서버 구조가 아니라, GitHub 관리 + Vercel 배포 + Supabase 저장 구조로 유지합니다.

## User Value

Donggri Ledger의 강점은 “지금 얼마를 썼는가”보다 “지금 내 돈 흐름이 어디로 가고 있는가”를 함께 보여준다는 점입니다.

예를 들어:

- 카드로 3월에 결제했지만 실제 출금은 4월인 지출
- 매달 반복되는 통신비, 구독료, 월세
- 예산 초과 위험이 보이는 카테고리
- 생활비와 함께 봐야 의미가 생기는 투자 계좌

이런 흐름을 따로따로 보지 않고 한 서비스 안에서 연결해 보여주는 것이 제품의 핵심입니다.
